v1.4.0 Latest May 2026
New Features
  • Rich clipboard — pasting from Google Docs, web browsers, and other apps now preserves bold, italic, headings, bullet lists, numbered lists, and tables. Tables are embedded as read-only blocks.
  • Format painter — tap the brush icon in the toolbar to copy formatting from selected text and apply it to another selection (macOS + Android).
  • Strikethrough — available in the Text Style group on all platforms.
  • Android back navigation — the back gesture now follows the panel hierarchy: editor → notes list → folder sidebar → exit to home.
  • Android text magnifier — appears during selection handle drag for precise cursor placement.
  • macOS triple-click — selects the full paragraph, not just the line.
  • macOS Edit menu — Paste and Match Style, Delete, Find, Copy Formatting, and Cancel Formatting added to the native Edit menu.
  • Drive storage warnings — a warning banner appears when your Google Drive storage reaches 90% full (once per session), and a persistent alert shows on every sync attempt when storage is completely full.
  • Settings About section — the app version number and a link to the website are now shown at the bottom of the Settings screen.
  • App renamed to My Notes — the name is updated across all platforms.
Bug Fixes
  • Android: heading and font-size sub-menus now open correctly when the keyboard is visible.
  • Android: fixed a 401 authentication error caused by Play Services returning a stale OAuth grant.
  • macOS: fixed a 403 insufficient-scope error when restoring a session without Drive access.
  • macOS: triple-click paragraph selection now works correctly on headings.
  • macOS: set a minimum window size to prevent layout overflow.
  • Fixed multi-note push loss when switching notes rapidly.
  • Drive sync icon now correctly reflects the current sync state.
  • Empty new notes are automatically discarded when switching away.
Version 1.3.0 Latest release May 2026
New Features
  • Format Painter — copy text formatting from one selection and apply it to another with a single gesture. Available via the paint-brush icon in the toolbar and via Edit → Copy Formatting on macOS.
  • Strikethrough — added to the Text Style group on Android and to the inline toolbar on macOS.
  • Auto-discard empty notes — switching away from a newly created note without typing anything removes it automatically.
  • Android magnifier — the system magnifier now appears during text selection handle drag.
macOS
  • Full native Edit menu: Undo, Redo, Cut, Copy, Copy Formatting, Paste, Paste and Match Style, Delete, Select All, Find.
  • Triple-click selects the entire paragraph under the cursor.
Android
  • Toolbar sheet preserves keyboard focus and adds a drag handle for easier dismissal.
Bug Fixes
  • All edited notes are now pushed to Drive when switching notes quickly via the + button.
Version 1.2.0 May 2026
New Features
  • Hyperlinks — insert, edit, and open links from within the editor. Right-click linked text for quick editing on macOS; long-press on Android.
  • Grouped toolbar — bottom-sheet overflow for the toolbar on Android; compact inline layout on macOS. Includes bundled decorative fonts (Nunito, Pacifico, Roboto Mono, Ibarra Real Nova, Square Peg).
  • Account deletion — permanently delete your account and all data (local and Drive) from within the app. Cannot be undone.
  • Privacy Policy page — accessible from the app footer and the website.
Android
  • Release APK with signing configuration for distribution outside the Play Store.
  • Drag-and-drop move-to-folder enabled on wide (tablet) Android layouts.
  • Drawer closes automatically when selecting a folder.
macOS
  • Native context menu for the editor (right-click): Insert Link, Edit Link, Delete Note.
  • Cmd+click to open links in the browser.
Bug Fixes
  • Title changes now debounce at 5 s (was 15 s) to push quickly to Drive.
  • Note content debounce reduced to 10 s (was 15 s).
  • Transient network errors no longer flip the sync icon to the error state.
  • Full-word link removal when selection is partial.
Version 1.1.0 April 2026
New Features
  • Move notes to folders — drag a note onto a folder in the sidebar (macOS), or long-press and choose Move to Folder (Android).
  • Log-based incremental sync — sync now uses an append-only log on Drive. Only changed items are downloaded; poll overhead is negligible when nothing has changed.
  • Settings screen — access via the user menu at the bottom of the sidebar.
  • Structured logging — all app events are written to a local log file for easier debugging.
Sync
  • App folder ID is cached to reduce API calls.
  • First-sync note downloads are parallelised for faster initial load.
  • Concurrent poll cycles are serialised to prevent duplicate downloads.
  • Batch move uploads — multiple moves within 5 s are sent in a single Drive write.
  • 30-second timeout on all Drive API requests.
Bug Fixes
  • Default view on login is All Notes.
  • Tab key now inserts a fixed-width indent and advances the cursor correctly.
  • Push operations are serialised to prevent sync_log.json race conditions.
Version 1.0.0 April 2026
Initial Release
  • Rich text editor — bold, italic, underline, strikethrough, font family & size, text colour, highlight, headings, lists, checklists, indent, undo/redo.
  • Images — insert from gallery, drag and drop from Finder (macOS), or paste from clipboard. Right-click to delete.
  • Folders — create, rename, delete. Deleting a folder moves its notes to the inbox.
  • Google Sign-In — silent sign-in restores sessions automatically on restart.
  • Local account — username and password stored on-device; no Google account required.
  • End-to-end encryption — notes are encrypted with AES-256-GCM before leaving the device. Your key lives only in your own Google Drive.
  • Google Drive sync — notes and folders sync automatically across devices.
  • Search — full-text search across all notes by title and content.
  • Available on macOS and Android.
